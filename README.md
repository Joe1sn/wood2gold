# wood2gold

Minecraft 1.15.x 稀有矿物合成辅助的forge模组

主要是自己不想挖矿去肝，所以写了个合成钻石、黄金、鞘翅和烟花等的mod来节约时间

个人使用1.15.2开发，按道理说是1.15.x通用的

直接把jar文件丢到mods文件夹下就可以了
